# WordSearch

WordSearch is a modern mobile word search game built with Flutter and Dart. It is designed as an Android-first project with a polished mobile UI, generated puzzle boards, and swipe-based word selection.

## Current Features

- Custom mobile-first home screen
- Multiple difficulty modes
- Generated word search boards with filler letters
- Swipe and drag word selection
- Restartable puzzles
- Found-word and completion tracking

## Tech Stack

- Flutter
- Dart
- Material 3

## Project Goal

The goal of this project is to turn WordSearch into a clean, modern puzzle app that feels good on phones and is ready to grow toward a Google Play release.

## Getting Started

1. Install the Flutter SDK.
2. Set up Android Studio with the Android SDK and an emulator.
3. Run `flutter pub get`.
4. Run `flutter run`.
