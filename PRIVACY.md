# Privacy Policy

WordTrail is designed as an offline-first word-search game.

## Data Collection

WordTrail does not collect, sell, or share personal data.

The app does not require an account and does not request access to location, contacts, camera, microphone, photos, or files.

## Local Progress

WordTrail saves gameplay progress on the device so players can keep streaks, best times, cleared boards, selected settings, and recent puzzle history between app sessions.

This saved progress stays on the device. It is not uploaded to a server by this app.

## Device Permissions

The release app is intended to run without sensitive Android permissions. Debug builds may use normal Flutter and Android development tooling while testing locally.

## Future Changes

If online features, analytics, ads, purchases, reminders, or cloud saves are added later, this policy should be updated before those features are released.
